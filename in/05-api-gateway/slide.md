## Проблема

Фронт и микросервисы ломались из‑за **неявных изменений API** без версионирования.

## Инструменты

- **OpenAPI** — контракты
- **Spectral** — линт спецификаций
- **CI** — проверка на PR

## Решение

Единый gateway + **обязательный diff OpenAPI** в ревью. Breaking change — только с версией v2.

![Схема gateway](scheme.svg)

## Демо

Пример diff контракта и падение CI при несовместимости.
