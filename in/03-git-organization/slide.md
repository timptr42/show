## Проблема

Проекты разъехались по личным форкам: **непонятно**, куда коммитить демо и эксперименты с ботами.

## Инструменты

- GitHub org **CPL-team / CPL_projects**
- Отдельные «пространства» под направления
- Теги и README в корне каждого пространства

## Решение

Договорились: **один org — несколько spaces**, в названии — направление (например, Alert-bot CURSOR).

![Структура репозитория](scheme-git-spaces.svg)

## Демо

Покажем карту каталогов и как завести новый блок выступления по тому же шаблону `in/` → `out/`.
