# HTML-презентация для выступления

Автосборка из каталога `in/` в `out/`.

## Быстрый старт

```bash
pip install -r requirements.txt
python build.py
```

Откройте `out/index.html` в браузере, нажмите **F11** для полноэкранного режима.

**Онлайн:** https://timptr42.github.io/show/

Навигация: клик по правой половине экрана / кликер — вперёд, левая половина — назад. Стрелки на клавиатуре тоже работают.

## Структура `in/`

| Папка | Назначение |
|-------|------------|
| `00-welcome` | Заглушка: фоновые видео (без звука), заголовок из frontmatter |
| `01-…` … `NN-…` | Блоки выступления |
| `99-questions` | Финальный экран с вопросами |

В каждом блоке — файл **`slide.md`** с жёсткими секциями `##`.

### Блок выступления (`type: talk`)

```markdown
---
title: "Название блока"
author: "Имя · CPL"
type: talk
---

## Проблема
...

## Инструменты
...

## Решение
...
![схема](scheme.svg)

## Демо
Краткий текст перед показом медиа.

## Вопросы?
...
```

Медиа для демо: файлы `demo-*` или любые изображения/видео в папке (кроме упомянутых в MD схем). Показ **последовательный** — один файл на слайд.

### Заглушка и общие вопросы

```markdown
---
title: "Заголовок"
subtitle: "Подзаголовок"
type: welcome   # или questions
---

## Текст
Опционально для экрана questions.
```

Видео для фона: `bg-01.mp4`, `bg-02.mp4`, … в `00-welcome` и `99-questions`.

### Раскладка видео из welcome

Положите все ролики в `in/00-welcome/`, затем:

```bash
python organize_videos.py
python build.py
```

Скрипт разложит **по 2 видео** в каждый блок выступления (`demo-01`, `demo-02`), остальные — поровну в welcome и Q/A (`bg-01`, …). Имена kling из Downloads тоже подхватываются.

Если видео нет, сборщик создаст короткие заглушки только в `out/` (нужен `imageio`).

## Сборщик

- Полностью очищает `out/`
- Копирует медиа в `out/assets/<имя-папки>/`
- Генерирует один `out/index.html`
